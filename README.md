🧑‍🍳 懷舊小食堂 - 長輩專屬記憶力訓練遊戲

這是一個專為「熟齡族群 / 長輩」設計的無壓力記憶力訓練小遊戲。
結合了日常生活的「買菜」情境，幫助長輩在充滿成就感的氛圍中活化大腦。

✨ 遊戲特色 (Vibe)

👵 友善的長輩介面：超大字體、高對比顏色，並使用直覺的日常食物 Emoji（🍎🥦🥩）取代繁雜文字。

❤️ 零挫折感設計：沒有滴答作響的倒數計時器，答錯了也不會「Game Over」，而是給予溫暖的語音鼓勵。

🔊 溫暖的聽覺回饋：內建親切的阿嬤語音提示與清脆的過關音效，增加遊戲沉浸感與陪伴感。

📱 跨平台支援：純前端網頁技術打造（HTML5 + Tailwind CSS），免下載安裝，手機、平板、電腦打開網址就能玩！

🎮 怎麼玩？

記憶菜單：畫面會短暫顯示今天的「採買清單」（如：蘋果、花椰菜、雞肉），請長輩試著記住它們。

市場挑選：清單收起後，會出現滿滿的菜市場網格，請從裡面挑出剛剛清單上的食材。

過關升級：全數找齊即可過關，隨著關卡推進，需要記憶的食材數量會慢慢增加喔！
用心為長輩打造的科技小玩具，歡迎分享給身邊的阿公阿嬤！
